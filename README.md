## 使用说明

拖入文件即可使用，软件自动常驻任务栏，悬停鼠标于右上角图标可获得提示。

点击🛠️后将添加“右键菜单-发送到”，支持不打开直接使用。

如果右键菜单失效，请右键点击🛠。

右下角⚓/🔱可设置置顶与否。

------
